Klasa ThreadPool

Czym jest ThreadPool

  1. Pula wątków zarządza dwoma rodzajami wątków
  2. Worker Threads : roboczy wątek
  3. I/O completion port thread : wykonują one operację wejścia/wyjścia np. dysk

Czym jest ThreadPool

  1. Używay przez WCF, ASP.NET/ASP.NET Core, TPL, PLinq (w sumie cały .NET)
  2. Nie możesz tworzyć swojej puli
  3. Posiada mechanizm trzymania "dobrej" liczby wątków
